## ❓ Problem Statement
Manual diagnosis of brain tumors from MRI images is time-consuming and requires expert radiologists.  
This project aims to develop an automated, accurate, and explainable deep learning system to classify brain MRI images into four tumor categories, assisting medical diagnosis.

---

## 🧠 Models Used
- **ResNet18** (Baseline CNN with residual connections)
- **EfficientNet-B0** (Efficiently scaled CNN for higher accuracy)
